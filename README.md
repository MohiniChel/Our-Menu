# This is a todo app with CRUD operation.

Functionalities:-

1. Have a list of food items with the following details 
 * Name
 * Price
 * Details
 * Image
 
2. Can toggle between the menu list
 * All
 * Lunch
 * Breakfast
 * Shakes
  
#### Used Technologies : Reactjs, React Hook.
